# QR Code Generator

A simple Python project that generates QR codes from text or URLs using the `qrcode` library.

## Features

- Generate QR codes from any text or URL
- Save QR code as PNG image
- Supports custom colors
- Easy to use and beginner friendly

## Technologies Used

- Python 3
- qrcode
- Pillow

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/QR-Code-Generator.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```

## Example

Input

```
https://www.linkedin.com/in/abhishek-s-gupta
```

Output

```
linkedin_qr2.png
```
